# localmusic

Local Music Player.

Flutter 框架， 使用 audioplayers audio_service 音频播放。
选择本地音乐播放，支持歌单和播放模式。
其他端已编译 可参考  https://localplayer.freeonetool.org/
MAC 和IOS暂时没编译 release版本，暂时买不起^_^
